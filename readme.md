## Features

- Add, update and remove the tasks
- Save the tasks in the localstorage


## Tech

- React
- NodeJs
- Flowbite
- Tailwindcss
- TypeScript
- Vite
- Vitest
- React icons

## Installation

Requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies **"npm i"** before run any of this next commands.

For run on development:
```sh
npm run dev
```

For build the project:
```sh
npm run build
```

For run the build:
```sh
npm run preview
```

For lint:
```sh
npm run lint
```

For fix lint problems:
```sh
npm run lint:fix
```

For run test:
```sh
npm run test
```

For run coverage:
```sh
npm run coverage
```
